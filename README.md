# Sol

Monorepo that houses cloud applications directly correlated to project.

**Utilizes:**
- Typescript
- GraphQL
- React
- Golang
- GRPC
- Protobufs
- Docker
- Terraform

# Technical Onboarding
1. Invite to Github organization
2. Invite to Asana
3. Invite to Dropbox
4. Create IAM user. Change password, setup MFA.
